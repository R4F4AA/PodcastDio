# Processo de Engenharia de Prompt: Do Roteiro à Entrega

Este documento detalha todos os prompts usados na criação do podcast "Do Zero ao Insight", desde a ideia inicial até os ativos finais, demonstrando um fluxo de trabalho completo com IAs Generativas (ChatGPT e Gemini).

---

## Fase 1: Concepção e Roteirização (com ChatGPT)

Nesta fase, o objetivo foi criar a base de conteúdo do podcast, definindo o tema, título e roteiro.

### 1. Brainstorming de Títulos
O primeiro passo foi definir um título de episódio que gerasse curiosidade para o público-alvo.
> **Prompt:** "Atue como host de podcast voltado para profissionais de dados iniciantes. Sugira 10 títulos de episódio que gerem curiosidade. Vamos seguir o seguinte background: Dados & Inteligência Artificial para Iniciantes. A ideia é abordar a área de dados para iniciantes que não conhecem ou estão iniciando na área de dados."

### 2. Estruturação do Roteiro Base
Após escolher o título, o foco foi estruturar o conteúdo central do roteiro.
> **Prompt:** "Vamos seguir com Do Zero ao Insight: Como Começar na Área de Dados sem Medo. Me ajude a estruturar o roteiro trazendo informações reais e referências para consulta."

### 3. Desenvolvimento de Seções Específicas
Foram usados prompts direcionados para construir partes-chave do roteiro.
> **Prompt (Seção 1):** "Contextualizar tipos de perfil e áreas dentro de dados (analista de dados, engenharia de dados, cientista de dados) e que para iniciantes o foco costuma ser ‘analista de dados’ ou ‘business intelligence’."
>
> **Prompt (Seção 2):** "4. Passos Práticos para Iniciar Hoje Mesmo — me ajude com o texto para essa parte."

*(**Resultado:** O roteiro detalhado que foi usado como base para a próxima fase.)*

---

## Fase 2: Refino, Identidade Visual e Documentação (com Gemini)

Com o roteiro base pronto, esta fase focou em otimizar o material para produção (áudio e visual) e criar os ativos de entrega para o portfólio.

### 4. Refino e Otimização para Voz de IA (Formato Alegre)
O roteiro foi adaptado para soar mais natural e dinâmico para uma narração de IA (como ElevenLabs), focando em um tom específico.
> **Prompt:** "Tenho este roteiro [roteiro original do ChatGPT fornecido]. Me ajude a melhorá-lo. Leve em consideração que usarei IAs gratuitas para produzir o podcast. Quero também que o tom do podcast seja alegre."

### 5. Reestruturação Criativa (Formato "Ei Nerd")
Para criar uma versão alternativa e mais curta, foi usada uma estrutura de prompt com regras rígidas de formato e estilo.
> **Prompt:** "Você é um roteirista de podcast. Vamos criar um roteiro... O formato deve ser [INTRODUÇÃO], [CURIOSIDADE 1], [CURIOSIDADE 2], [FINALIZAÇÃO]. No bloco [INTRODUÇÃO] substituir por uma introdução igual às dos vídeos do canal 'ei nerd'..." *(e todas as outras regras definidas)*.

### 6. Geração da Capa (Identidade Visual)
Foi solicitado a criação da arte de capa para o podcast, definindo o estilo e os elementos visuais.
> **Prompt:** "Quero que gere uma imagem para servir de capa para o podcast 'Do Zero ao Insight'. O tema é 'Como Começar na Área de Dados sem Medo', e o público-alvo são iniciantes. O tom deve ser alegre, acessível e moderno..."

### 7. Criação da Documentação (README)
Para finalizar o portfólio, foi solicitado o arquivo de documentação do projeto, com ênfase no propósito didático.
> **Prompt:** "Com base no que ajustamos em relação ao podcast, quero estruturar um README. É muito importante que este README frize que o projeto é apenas para fins didáticos."

### 8. Organização Final dos Prompts (Este documento)
Por fim, foi solicitado que todo o processo fosse compilado e organizado para a entrega do desafio.
> **Prompt:** "Consegue me trazer os pronpts que usei organizados para postar na entrega do desafio?"
